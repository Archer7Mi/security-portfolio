# Policies

This folder is for **portfolio-safe** policy and GRC artefacts.

The point isn’t to be fancy — it’s to show that I can write clear, usable security docs and connect them to real operations.

## What I’m adding here
- Incident Response policy template
- Risk register template (simple and readable)
- Later: access control, data classification, acceptable use

## What it demonstrates
- Clear documentation (scope, owners, review cycles)
- Basic risk thinking (likelihood/impact and mitigations)
- How governance supports SOC work (reporting, escalation, recovery)
