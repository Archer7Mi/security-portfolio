# Security Portfolio

This repo is my **offline-first** security portfolio.

I’m building it so a reviewer can understand how I think in SOC / detection / GRC **without needing me to run a VM or a live tool**. Everything here is documentation, queries, and writeups that I can keep improving over time.

## Sections
- [SC-200](SC-200/): notes + artefacts aligned to Microsoft SC-200 (KQL, hunting, IR writeups).
- [Splunk](Splunk/): SPL detections/hunts + tuning notes and dashboard placeholders.
- [ISO27001](ISO27001/): GRC artefacts (SoA template, audit-style docs, risk thinking).
- [CaseStudies](CaseStudies/): incident/detection case studies with evidence placeholders.
- [Policies](Policies/): policy templates and a portfolio-safe risk register.

## How I use this repo
- Each section has a short README describing what I’m adding and why.
- Case studies include placeholders for screenshots/log snippets so I can fill them later.
- I keep examples **redacted and safe for public sharing**.

## Quick start (what I’ll add next)

- a filled-out sample SoA subset + a risk register with realistic mitigations
