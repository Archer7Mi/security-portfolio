- KQL fundamentals (filtering, joins, time windows, summarization)
- clear documentation (so someone else could follow my steps)

## How it ties to SOC / governance / risk

- Governance: repeatable process and documentation discipline
- Risk: prioritizing investigations and detections based on impact
